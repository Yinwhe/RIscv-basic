# 指令格式

**RV32I有6种指令格式，且所有指令都是32位长**

- 用于寄存器-寄存器操作的 **R 类型指令**
- 用于短立即数和访存 load 操作的 **I** **型指令**
- 用于访存 store 操作的 **S** **型指令**
- 用于条件跳转操作的 **B** **类型指令**
- 用于长立即数的 **U** **型指令**
- 用于无条件跳转的 **J** **型指令**

- 所有位全部是 0 或全是1 的指令是非法的 RV32I 指令。因此, 试图跳转到被清零的内存区域的错误跳转将会立即触发异常。
- 指令都是按4字节进行对齐的，如果跳转时的指令没有对齐，会在跳转指令上报错
